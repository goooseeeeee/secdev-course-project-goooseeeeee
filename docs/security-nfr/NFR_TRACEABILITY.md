## Примечания

- **AUTH-01, AUTH-02** — задачи из блока аутентификации пользователей.
- **PERF-01** — относится к производительности интерфейса списка желаемых вещей.
- **SEC-01** — автоматический аудит зависимостей (например, через `npm audit` или `dependabot`).
- **OPS-02** — DevOps-задача по управлению секретами и ключами API.
- **LOG-01** — отвечает за централизованный сбор логов для отслеживания ошибок.
- **API-03** — защита от DDoS/спама, ограничение количества запросов в минуту.
- **TEST-05** — обеспечивает устойчивость и надёжность кода.
